# 🏦 Bank Management System

A **console-based Bank Management System** developed in **C++** using **Object-Oriented Programming (OOP)** and **File Handling**. This project simulates core banking operations such as account creation, deposits, withdrawals, balance inquiries, and customer record management through a simple menu-driven interface.

---

## 📌 Project Overview

The Bank Management System is designed to help users manage bank accounts efficiently. Customer data is stored permanently using file handling, ensuring records remain available even after the application is closed.

This project demonstrates practical implementation of OOP concepts, file handling, and modular programming in C++.

---

## ✨ Features

- 🆕 Create New Bank Account
- 💰 Deposit Money
- 💸 Withdraw Money
- 💳 Check Account Balance
- 🔍 Search Account by Account Number
- 📋 Display All Accounts
- ✏️ Update Customer Details
- 🗑️ Delete Account
- 📊 Display Total Number of Accounts
- 💾 Persistent Data Storage using File Handling
- 📂 Menu-Driven Console Interface

---

## 🛠️ Technologies Used

- **Language:** C++
- **Programming Paradigm:** Object-Oriented Programming (OOP)
- **File Storage:** File Handling (`fstream`)
- **Compiler:** g++ (MSYS2/MinGW)
- **IDE:** Visual Studio Code

## 🚀 How to Run

### 1️⃣ Clone the Repository

```bash
git clone <repository-url>
```

### 2️⃣ Open the Project

Open the project folder in Visual Studio Code.

### 3️⃣ Compile

```bash
g++ main.cpp BankAccount.cpp FileHandler.cpp -o BankManagementSystem.exe
```

### 4️⃣ Run

```bash
.\BankManagementSystem.exe
```
